libCubeEye  A cross-platform user library to access ToF camera.
How to Install CubeEye Python Package

Install Python Packages.

CubeEye supports Python 3.8 version.
You can use your own Python 3.8 or use one that is installed in xxx-xxx-xxx\thirdparty\Python-3.8.18\python.exe

Note
In case of Linux, Python will be found in xxx-xxx-xxx/thirdparty/python/bin/python3

Install Basic Packages

Install basic packages to run our CubeEye camera and examples.
OpenCV and Open3D are optional only to demonstrate our CubeEye camera

$ cd xxx-xxx-xxx\release\python
$ ..\thirdparty\Python-3.8.18\python.exe -m pip install --upgrade pip
$ ..\thirdparty\Python-3.8.18\python.exe -m pip install --upgrade numpy
$ ..\thirdparty\Python-3.8.18\python.exe -m pip install opencv-python
$ ..\thirdparty\Python-3.8.18\python.exe -m pip install open3d


Build CubeEye Package

You can build CubeEye Package in order to use and distribute to other environment

$ ..\thirdparty\Python-3.8.18\python.exe -m pip install --upgrade build
$ ..\thirdparty\Python-3.8.18\python.exe -m build


Install CubeEye Package

To use CubeEye, you must install CubeEye package to your python environment.
You can install it from the directory or built package.

$ ..\thirdparty\Python-3.8.18\python.exe -m pip install .

Or

$ ..\thirdparty\Python-3.8.18\python.exe -m pip install dist\CubeEye-X.X.X.tar.gz
