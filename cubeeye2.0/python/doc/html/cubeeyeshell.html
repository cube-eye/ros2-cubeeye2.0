libCubeEye  A cross-platform user library to access ToF camera.
How to run CubeEye Examples

Simple testing ToF Camera with Examples

Note
For linux, please source setup.bash file for other shared libraries
$ cd xxx-xxx-xxx\release\python\examples
$ source setup.bash

Run CubeEye example

$ cd xxx-xxx-xxx\release\python\examples
$ ..\..\thirdparty\Python-3.8.18\python.exe example.py


Usage: [COMMAND]... [OPTION]...

Usage: >> [COMMAND]... [OPTION]
search searching for ToF camera device. - e.g.: >> search
select select camera device in the searched list. - e.g.: >> select 0
run run the selected camera device with frame type. - e.g.: >> run 6
stop stop the camera device. - e.g.: >> stop
set set property. - e.g.: >> set framerate 8u 7
get get property. - e.g.: >> get framerate
quit quit this program. - e.g.: >> quit
