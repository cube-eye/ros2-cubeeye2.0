libCubeEye  A cross-platform user library to access ToF camera.
How to use CubeEyeShell

Simple testing ToF Camera with CubeEyeShell


Installing required packages (Ubuntu 16.04)

$ sudo apt-get install libgtk2.0-dev libgtk-3-dev


Usage: [COMMAND]... [OPTION]...

search searching for ToF camera device
select select camera device in the searched list
run run the selected camera device & You can see the image frames
stop stop the camera device
get get property from selected camera device
set set property to selected camera device
deselect release the selected camera device
history display manipulate the history list
version display last released version
quit release the selected camera device & exit
help display this help
Examples:
$ CubeEyeShell> search
$ CubeEyeShell> select 0
$ MDC100SF: run
$ MDC100SF: set amplitude_threshold_min 10
$ MDC100SF: set noise_filter1 on
$ MDC100SF: stop
$ MDC100SF: deselect
$ CubeEyeShell> quit
